Crystals Collector

At the start of the game, you will be given a random number and four crystals. Each crystal is worth a different amount of points. By clicking on the crystals below, try to match the random number to win.

The value of the crystals are hidden to the naked eye, so be careful of which ones you press!

The game will restart with each loss or win and the value of the crystals will change.

Good luck!
